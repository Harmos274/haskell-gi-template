# Gtk Quasi Quotation Syntax

```
MyWindow

<t: text>

<b: button>
```

| Pattern        | Meaning             |
|----------------|---------------------|
| `<t: text  :>` | Text displayer      |
| `<b: label :>` | Button with label   |
| `<?: label :>` | Checkbox with label |
